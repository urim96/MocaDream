## :bulb: 주요 기능
#### 일반 회원 기능
- 회원가입
- 로그인
- 아이디찾기, 비밀번호 찾기
- 회원정보 수정
- 회원 탈퇴
- __이용방법__
- 공지사항
- 오시는 길 (KAKAO API 사용)
- 방 정보
- __예약 하기__
- __예약 확인__
- __나의 전체 예약 내역 확인, 취소__
  
#### 관리자 기능
- 회원 조회, 경고 처리
- 방 등록(파일 업로드), 수정, 삭제
- 전체 예약 내역 조회(날짜 검색)
- 이용자 입실 처리 
- 실시간 예약 내역 조회
- __예약 취소__
- 공지사항 작성, 수정, 삭제
